# Number_plate_recognition

README Description
Automatic Number Plate Recognition (ANPR) System

A modular Automatic Number Plate Recognition (ANPR) system built from scratch using Python, YOLOv8, EasyOCR, and OpenCV. The project detects multiple vehicle number plates in real time, extracts the registration number using OCR, and logs the results into an Excel file.

The project follows a modular architecture with separate components for vehicle detection, OCR, data logging, and the main processing pipeline, making it easy to extend and maintain.

Current Features
🚗 Real-time number plate detection using YOLOv8
🔍 OCR using EasyOCR
📝 Excel-based logging of detected plates
🎯 Confidence threshold filtering
🖼️ Image preprocessing (Resize, Grayscale, CLAHE)
🔄 Duplicate detection filtering using time-based cache
🧩 Modular project structure
